Projeto Sindicato S.T.P.M.J.

Principais funções:
    - Tela de cadastro de sócio
    - Cadastro de empresa e parceiros (Para vales "Chequin")
    - Emissões personalizadas de relatórios
        - Filtros combinados utilizando qualquer campo da tela de cadastro.
        - Report de aniversário
    - Impressões:
        - Imprimir folha de cadastro, para assinatura.
        - Imprimir carteirinha de sócio com dependentes.
        - Imprimir "Chequin" (Vales)

*IMPORTANTE* DEVE SER FEITO BANCO DE DADOS ACCESS E DEVE SER NA NUVEM
